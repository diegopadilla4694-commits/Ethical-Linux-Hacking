#First Hack The Box Practice
This repository includes my notes throughout my progress in using Linux.

In this first practice at HTB Academy, I learned to use basic Linux commands to navigate Linux through a linear command-line interface.
I was able to access a virtual machine using the IP address assigned to me in the course. I used my own Linux OS with the following commands:
-ssh [username]@<IP>: to access the machine's address along with the user.

-whoami: Used to confirm the user htb[username].

-uname -r: Obtained the kernel version of the machine.

-ip addr: Searched for the interface (ens3) with MTU 1500.
cat /var/mail/htb[username]: Opened the email to look for clues.

-I used OpenVPN GUI to connect to the HTB Academy virtual machine.

-Note: I am not including the machine's IP address due to website rights.
